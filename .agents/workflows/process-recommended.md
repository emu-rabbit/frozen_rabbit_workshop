# 執行步驟

1. 讀取專案底下的 `src/data/recommended.json` 檔案。
2. 修正所有對json檔來說是錯誤的語法，使其成為一個正確可以解析的json檔
3. 尋找陣列中並分析所有的推薦筆記 (`Note` 物件)，特別檢查其 `name` 屬性。
4. 如果 `name` 是字串，請將他展開成物件。如果 `name` 已經是物件且包含 `tw` 屬性，則分析其 `tw` 內容。
5. 由於本筆記為 Final Fantasy XIV (FF14) 遊戲相關工具，對於裝備套裝名稱，請務必辨識名稱中的關鍵字並遵循以下對應規則進行翻譯，不得自行調整：

| 關鍵字 (TW) | 簡體中文 (cn) | 英文 (en) | 日文 (ja) |
| :--- | :--- | :--- | :--- |
| **禦敵** | 御敌 | Fending | タンク |
| **制敵** | 制敌 | Maiming | メレー |
| **強襲** | 强袭 | Striking | メレー |
| **游擊** | 游击 | Scouting | メレー |
| **精準** | 精准 | Aiming | レンジ |
| **詠咒** | 咏咒 | Casting | キャス |
| **治癒** | 治愈 | Healing | ヒラ |

**注意：** 在翻譯時，請僅替換關鍵字部分，並保留名稱中的其他前後綴資訊（例如等級、iLv、件數說明等）。

6. 將翻譯完成並更新後的內容，寫回 `src/data/recommended.json` 檔案中，請務必保持原本的 JSON 縮排與格式。
7. 最後請確保這份陣列中每個物件的格式都符合 `src/types/note.ts` 中 Note 的資料格式定義
8. 處理完成後，請整理一份清單向使用者回報你翻譯了哪些筆記以及其結果。
